Функции HabrAjax

Поддержка публикации комментариев на сервисе habraQuotes.ru
в скрипте HabrAjax


Юзерскрипт HabrAjax способен работать в браузерах Firefox 3.6-18+, Chrome16-22+, Opera10-12+ и работает на страницах сайта habrahabr.ru. Для Firefox требуется установить аддон (расширение) Scriptish или (менее желательно) GreaseMonkey.

Кратко.
Для быстрой публикации ссылок на интересные комментарии в HabrAjax имеется контекстная кнопка "HQ". Достаточно выделить любой текст в комментарии или охватить выделением текст из группы комментариев, навести мышь на контекстную кнопку " <_>", и появится несколько других кнопок, выполняющих разные функции. Если в настройках HabrAjax была установлена функция "Выделять и отправлять на HabraQuotes", среди них будет кнопка "HQ". По умолчанию, функция установлена. Нажатие "HQ" приведёт к выводу сообщений вверху страницы, при которых, если они будут сообщать об успешной отправке, на сервис HabraQuotes.ru отправится ссылка на выделенный комментарий.

Узнать о скрипте больше - страница хостинга HabrAjax   для работы с сайтом Habrahabr.ru.
На ней кнопка "Install" - установить скрипт.

Подробнее.
1. Если функция публикации в настройках HabrAjax не включена, установить выбор настройки для функции "Выделять и отправлять на HabraQuotes".

Настройки юзерскрипта на странице сайта
После изменения настроек - нажать "Сохранить".
2. Перезагрузить страницу или открыть другую, чтобы настройка начала действовать.
3. Выделить любой текст в комментарии или в группе комментариев. Под текстом появится едва заметная кнопка " <_>" - ворота для различных контекстных операций.

При наведении мыши на неё откроются все остальные кнопки (команды операций).
4. Выбираем кнопку "HQ", чтобы начать отправку ссылки на комментарий и делаем клик по кнопке.

В баре цитат и напоминаний вверху окна появится знак ожидания ответа от сервиса HabraQuotes (крутящиеся стрелки). Они не будут мешать дальнейшей работе с сайтом.
5. Результат может быть как успешным, если сервис habraquotes.ru работает нормально,

так и неуспешным по разным причинам, которые не детализируются:

Чтобы попытаться зайти на сайт habraquotes.ru и отправить комментарий с него, нужно нажать ссылку в сообщении об ошибке.
Возможные ошибки:
*) комментарий или их группа уже опубликованы ранее;
*) страница оказалась слишком большой и сложной для парсера;
*) сервис не работает;
*) сервис перешёл на другой протокол работы;
*) связь с интернетом, сервисом или Хабром разорвалась.

При успешной публикации на адресе http://habraquotes.ru/fresh/ обнаружится такая группа сообщений:


Публикация - анонимная; имя пользователя не отправляется на сервис (и не будет), даже если имя отправителя присутствует на странице.

Как начать пользоваться?

1. Установить аддон Scriptish для юзерскриптов (Firefox); для Chrome и Opera аддонов не требуется, имеется своя процедура установки юзерскрипта - для Chrome, для Opera.
2. Установить юзерскрипт HabrAjax для работы с сайтом Habrahabr.ru .
